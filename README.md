# Customer-Purchase-and-Shopping-Behavior-Analysis
Data Analytics project showcasing customber behavior analysis using python,sql and power Bi

## Overview

This project focuses on analyzing customer shopping behavior to uncover meaningful insights that can help retail businesses improve sales performance, customer engagement, and marketing strategies. The analysis explores purchasing trends across demographics, product categories, discounts, payment methods, and sales channels (online vs. offline). The project follows a complete data analytics lifecycleâ€”from data preparation to visualization and business reporting.

 ## Dataset

The dataset contains customer-level and transaction-level information related to shopping behavior, including:

Customer demographics

Product categories

Purchase frequency and amounts

Discounts and offers

Payment methods

Sales channels (online/offline)

Ratings and reviews

Seasonal purchase trends
The dataset was loaded into Python for analysis and later stored in relational databases for structured querying.

 ## Tools & Technologies

Python (Pandas, NumPy, Matplotlib, Seaborn)

SQL (PostgreSQL / MySQL / SQL Server)

Power BI (Data visualization and dashboarding)

pgAdmin / MySQL Workbench / SQL Server Management Studio

 # Project Workflow / Steps

### 1.Data Loading

Imported raw dataset into Python using Pandas.

Performed initial data inspection and validation.

### 2.Exploratory Data Analysis (EDA)

Analyzed customer demographics and purchase patterns.

Identified trends in product categories, discounts, and seasons.

Studied customer behavior across online and offline channels.

### 3.Data Cleaning & Preprocessing

Handled missing values and duplicates.

Corrected data types and standardized column formats.

Prepared clean data for database storage and visualization.

### 4.Database Integration & SQL Analysis

Loaded cleaned data into PostgreSQL / MySQL / SQL Server.

Executed SQL queries to analyze:

Top-selling products

Customer segmentation

Payment preferences

Repeat purchase behavior

### 5.Power BI Dashboard

Built an interactive dashboard to visualize key metrics.

Included filters for demographics, time period, and sales channels.

Enabled easy interpretation for business stakeholders.

### 6.Reporting & Presentation

Created a detailed analytical report summarizing insights.

Designed a professional presentation using Gamma AI for stakeholder communication.

 ## Dashboard

The Power BI dashboard highlights:

Customer purchase trends

Sales performance by category and channel

Discount impact on buying behavior

Seasonal and regional patterns

Payment method preferences

The dashboard supports data-driven decision-making with interactive visuals and filters.

Identified key factors influencing customer purchase decisions.
